Aula introdutória

Nessa aula será apresentado detalhes mais técnicos de strings, sendo eles:

  • \n pula uma linha
  • Para imprimir aspas em uma string, deverá ser usado a aspa oposta do que foi utilizado para iniciar a mesma.
  • Pode ser feito lógica dentro de uma string com template literals
  • Pode ser feito concatenação de strings com "+"
