Aula introdutória

Nessa aula é apresentado como inserir Javascript no código HTML. De inicio, será abordado as formas de inserir dentro do próprio arquivo HTML, mas também, com a melhor prática, com um arquivo externo.

A maior parte do conteúdo será exibido no console, pois dados e informações serão exibidos pelo método console.log().

OBS: Usei css interno também por se tratar de aulas introdutórias.
